# Text-Classification-Application
文本分类应用,采用模型是Text-CNN
- 英文数据集：影评评论
- 中文数据集(已去除停用词)：旅客酒店住宿评论；网络购物评论；书评；对话语料。
原始语料以及预处理脚本请见：链接：https://share.weiyun.com/5UoZkUx 密码：6tnnxy
- 训练好的模型：链接：https://share.weiyun.com/5tYgD3F 密码：weumvg
